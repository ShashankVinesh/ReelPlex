
# Instagram Reel Downloader 📥

Download Instagram reels using a simple link.

## Features
- Paste reel link and download
- Fast API-based processing
- Simple UI

## Tech Stack
- Kotlin , XML , MVVM
- REST API integration

## How It Works
1. User pastes Instagram reel URL
2. App sends request to API
3. Returns downloadable video
